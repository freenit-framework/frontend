# Freenit Frontend

### [Tutorial](https://github.com/freenit-framework/frontend-tutorial)

### Shell Scripts
To star development:
```
bin/init.sh
bin/devel.sh
```

To run tests:
```
bin/init.sh
bin/test.sh
```

### CBSD/Reggae
To start development:
```
echo DEVEL_MODE=YES >vars.mk
make devel
```

To run tests:
```
echo DEVEL_MODE=YES >vars.mk
make test
```
